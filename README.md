# Todo App

A terminal-based Todo application built with Textual, providing a fast and interactive command-line interface for managing tasks.

---

## Requirements

- Python 3.10 or higher
- [uv](https://github.com/astral-sh/uv) installed

---

## Installation

1. Clone the repository:

```bash
git clone https://github.com/ikamania/todo-app.git
cd todo-app
```

2. Create environment and install dependencies:

```bash
uv sync
```

3. Run the application:

```bash
uv run python main.py
```
